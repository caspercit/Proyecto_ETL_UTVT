\# Proyecto ETL UTVT



\## Descripción

Proyecto ETL desarrollado en SQL Server Integration Services (SSIS) para cargar información de Clientes, Productos y Ventas hacia un Data Warehouse.



\## Tecnologías utilizadas

\- SQL Server

\- SQL Server Integration Services (SSIS)

\- Visual Studio 2022

\- Git y GitHub



\## Estructura



\- data/

\- database/

\- ssis/

\- docs/

\- evidencias/



\## Flujo del proceso



1\. Lectura de archivos CSV.

2\. Validación de datos.

3\. Carga de DimCliente.

4\. Carga de DimProducto.

5\. Carga de FactVentas.

6\. Registro en LogProceso.

7\. Manejo de errores.

8\. Movimiento y compresión de archivos.
